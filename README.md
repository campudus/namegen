namegen
=======

Feed it arrays and you get more or less funny nicknames.

Example Usage
-------------

    var nameGen = require('namegen.js');
    var names = [
      [ '', 'My', 'Old', 'The' ],
      [ '', 'Aggressive', 'Black', 'Blind' ],
      [ 'Ant', 'Athlete', 'Avenger' ]
    ];
    
    nameGen.fromArray(names);

See `js/example.js` for more examples, like german nicknames.
